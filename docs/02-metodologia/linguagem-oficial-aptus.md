# Linguagem Oficial da Aptus

Este documento define o vocabulário oficial da plataforma Aptus. O objetivo é manter consistência entre a metodologia, a interface do usuário e o desenvolvimento do sistema.

## Termos para o usuário

- Aluno
- Competência
- Habilidade
- Sessão de Desenvolvimento
- Grau de Preparo
- Grau de Autonomia (indicador principal)
- Competência Consolidada

## Termos técnicos (código)

- Session
- Competency
- Skill
- Progress
- Evolution
- History

## Diretrizes

- Utilizar sempre os termos da coluna 'Usuário' em telas, mensagens, relatórios e comunicações.
- Utilizar os termos técnicos apenas no código-fonte, banco de dados e APIs.
- O termo 'Aluno' permanece como padrão da plataforma por ser universal no segmento de CFC.
- 'Grau de Preparo' representa a evolução prática do aluno.
- 'Grau de Autonomia' é o principal indicador da metodologia Aptus.
- 'Competência Consolidada' substitui expressões como 'concluído' ou 'aprendido', reforçando a ideia de desenvolvimento contínuo.

## Princípio

A Aptus não mede apenas desempenho. Ela acompanha o preparo do aluno e o desenvolvimento gradual de sua autonomia como futuro condutor.
