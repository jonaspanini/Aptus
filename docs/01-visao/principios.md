# Princípios da Aptus

1. O aluno deve sentir evolução.
2. O instrutor deve registrar rapidamente.
3. O gestor deve tomar decisões com poucos indicadores.
4. A metodologia vale mais que a tecnologia.
5. Offline é obrigatório.
6. Motivar sempre é melhor do que punir.
7. Cada funcionalidade deve economizar tempo ou aumentar o valor percebido.
8. A Aptus deve funcionar na rotina real de um CFC, e não apenas em demonstrações.
9. Simplicidade é uma funcionalidade.
10. Antes de desenvolver, validar se resolve um problema real.
